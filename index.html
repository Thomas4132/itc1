Programme officiel

Jupyter

Sur chaque TP, clliquer sur le titre ou pour lancer Binder avec le TP ouvert.

L'environnement de développement utilisé dans cours est Jupyter. Vous pouvez soit le télécharger en local en suivant les instructions sur https://jupyter.org, soit utiliser une version en ligne :

  • Utiliser Jupyter avec Binder (serveur distant)
    Attention : les données ne sont pas sauvegardées sur Binder. Pensez à download/upload votre travail.
  • Utiliser Jupyter avec Jupyterlite (exécuté dans le navigateur)
    Jupyterlite se lance normalement plus rapidement que Binder et permet de sauvegarder les fichiers (dans le navigateur).
    Malheureusement, Jupyterlite est encore en développement et il y a des problèmes (avec Safari, avec les fonctions récursives...). Utiliser Binder si ça ne marche pas.
